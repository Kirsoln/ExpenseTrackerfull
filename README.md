## Описание программы
Expense Tracker — это простое GUI-приложение на Python для учёта личных расходов.

Программа умеет:
- добавлять расходы
- показывать их в таблице
- фильтровать по категории
- фильтровать по дате
- считать общую сумму
- сохранять данные в JSON
- загружать данные из JSON при запуске

## Как использовать
1. Запустите файл `expense_tracker.py`
2. Введите:
   - сумму
   - категорию
   - дату в формате YYYY-MM-DD
3. Нажмите кнопку **Добавить расход**
4. Чтобы найти расходы, введите категорию или даты для фильтра
5. Нажмите **Применить фильтр**
6. Чтобы снова показать все расходы, нажмите **Показать всё**

## Пример использования
Пример 1:
- Сумма: 500
- Категория: еда
- Дата: 2025-05-01

Пример 2:
- Сумма: 120
- Категория: транспорт
- Дата: 2025-05-02

## Проверка программы
Можно проверить такие случаи:
- ввести правильные данные
- ввести отрицательную сумму
- ввести текст вместо суммы
- ввести дату не в формате YYYY-MM-DD
- проверить фильтр по категории
- проверить фильтр по диапазону дат
